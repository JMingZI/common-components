常见正则表达式

姓名

```js
/^(_)(_$)[a-zA-Z0-9_\u4e00-\u9fa5]+$/.test(name);
```
手机号

```js
/^(((13[0-9]{1})|(15[0-9]{1})|(17[0-9]{1})|(18[0-9]{1})|(14[0-9]{1}))+\d{8})$/.test(data.mobile);
```

邮箱

```js
/^\w+((-\w+)|(\.\w+))*\@[A-Za-z0-9]+((\.|-)[A-Za-z0-9]+)*\.[A-Za-z0-9]+$/.test(data.applicantMail);
```

验证码

```js
/^\d{4}$/.test(data.mobileAuthCode);
```
